# موكب العباس كفيل زينب — تطبيق الأندرويد

## ⚠️ خطوة واحدة مطلوبة قبل أول Build (مهم جداً)

بيئة إنشاء هذا المشروع لا تملك اتصال إنترنت، لذلك تعذّر إرفاق ملف
`gradle/wrapper/gradle-wrapper.jar` (وهو ملف ثنائي/binary وليس نصياً، ولا يمكن
توليده يدوياً بدون تحميله من خوادم Gradle الرسمية). كل بقية ملفات المشروع
كاملة وجاهزة تماماً.

اختر إحدى الطريقتين التاليتين (كلاهما يستغرق أقل من دقيقة):

### الطريقة 1 — الأسهل (داخل Android Studio)
1. افتح المشروع في Android Studio (Open an existing project).
2. إذا ظهرت رسالة تفيد بأن الـ Gradle Wrapper غير مكتمل، اضغط
   "OK" / "Fix Gradle Wrapper" ليقوم Android Studio بتنزيل الملف تلقائياً.
3. اضغط Sync Now، ثم Build > Generate Signed Bundle / APK.

### الطريقة 2 — عبر سطر الأوامر (إذا كان لديك Gradle مثبت على جهازك)
داخل مجلد المشروع، نفّذ الأمر التالي مرة واحدة فقط:

```
gradle wrapper --gradle-version 8.7
```

سيقوم هذا الأمر بإنشاء ملف `gradle-wrapper.jar` الصحيح تلقائياً، وبعدها
يمكنك استخدام `./gradlew` بشكل طبيعي في كل مرة.

## ما تم بناؤه في هذه النسخة

- **الهيكل العام**: شريط تنقل سفلي (Bottom Navigation) بثلاث تبويبات:
  الرئيسية، المنشورات، البث المباشر.
- **الرئيسية**: اسم الموكب (موكب العباس كفيل زينب)، الموقع (العمود 212)،
  وزر يفتح رابط خرائط جوجل مباشرة:
  https://maps.app.goo.gl/e1vXpv8Jf7HMFts17
- **المنشورات**: نشر منشور نصي، إعجاب، تعليقات (Bottom Sheet)، وحذف
  للمشرف (isAdmin) — البيانات محفوظة في الذاكرة حالياً (بدون سيرفر خلفي).
- **البث المباشر**: عرض روابط بث خارجية جاهزة (يوتيوب) داخل مشغل ويب،
  مع إمكانية إضافة/حذف الروابط من واجهة الإدارة.
- **الهوية البصرية**: ثيم Material 3 بألوان (أسود، ذهبي، أخضر داكن)
  مع دعم الوضع الليلي تلقائياً حسب إعداد الجهاز.
- **الأيقونة**: قربة ذهبية وهلال على خلفية حمراء داكنة (Adaptive Icon +
  نسخ PNG احتياطية لجميع الكثافات ولإصدارات أندرويد الأقدم من 8).

## الميزات غير المُدرجة بعد في هذه النسخة (تحتاج بنية تحتية/بيانات خارجية)

هذه الميزات ذُكرت في الطلب الأصلي لكنها تحتاج خدمات خارجية حقيقية
(سيرفر، مفاتيح API، أو محتوى ديني ضخم يجب مصدره بدقة)، ولذلك لم تُبنَ
في هذه النسخة الأولى تجنباً لتسليم كود وهمي/placeholder:

- بث مباشر حقيقي (RTMP) عبر سيرفر خاص بالتطبيق.
- التقويم الهجري ومواقيت الصلاة حسب GPS (يحتاج مكتبة حساب فلكي حقيقية).
- المصحف الشريف الكامل ومكتبة الأدعية والزيارات.
- السبحة الإلكترونية.
- شات داخلي للخدام + شات دعم للزوار (يحتاج قاعدة بيانات/سيرفر Realtime).
- تسجيل الدخول، الباركود، وقسم المفقودات بالصور.

يمكن إضافة أي من هذه الميزات في نسخة تالية دفعة واحدة أو على مراحل،
بحسب أولوياتك.
